# Rust for Python Developers

Following the Microsoft RustTraining python-book. Covers foundations through to a capstone CLI project.

The book is structured for developers coming from Python — it draws parallels between Python and Rust concepts throughout, making the transition intuitive.

## Phases
1. **Foundations** — Types, control flow, data structures, enums
2. **Core Concepts** — Ownership, modules, error handling, traits, closures
3. **Advanced Topics** — Concurrency, unsafe, FFI, migration patterns, best practices
4. **Capstone** — CLI Task Manager project
